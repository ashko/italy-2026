# איטליה 2026 🇮🇹

אפליקציית טיול (PWA) למשפחת ארז — כל הפרטים של הטיול לאיטליה במקום אחד:
מסלול יום-יום, לינה, כרטיסים, טלפונים, טיפים ורשימת ציוד. עובד גם **ללא אינטרנט**
וניתן להתקנה למסך הבית של הטלפון.

## תכונות
- 📅 מסלול מלא יום-יום (13 ימים, 13–25 באוגוסט 2026)
- 🛏️ לינה — מילאנו · אגם גארדה · דולומיטים · ורונה
- 🎫 כרטיסים — טיסות אל על + השכרת רכב Europcar
- 🧭 קישורי ניווט (Google Maps) לכל יעד
- ☎️ טלפונים חשובים בלחיצה אחת
- ✅ רשימת ציוד נשמרת במכשיר
- 📶 עובד אופליין (Service Worker) + התקנה כאפליקציה (PWA)
- 🔤 עברית מלאה (RTL)

## הרצה מקומית
```bash
python -m http.server 8000
# פתחו http://localhost:8000
```

## טכנולוגיה
HTML/CSS/JS טהור, ללא שלב build. כל הנתונים נמצאים בקובץ אחד: `js/data.js`.

## עדכון פרטים
לעריכת המסלול/הלינה/הכרטיסים — ערכו את `js/data.js` בלבד.
לאחר עדכון, העלו מחדש (push) וההגדרה תתעדכן באתר.
